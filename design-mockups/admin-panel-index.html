Endorfin · Admin panel

Four directions.

Same brand foundation (jet · signal · bone · Oswald · Poppins). Four different opinions on hierarchy, density, and primary surface. Pick one — or mix.

V1 Lowest risk · highest reuse

Heritage Editorial

Closest to existing super-admin. Sidebar nav, tables, pill tabs. Trust through familiarity. Drop-in compatible with current components.

  • · Sidebar navigation (proven on web)
  • · Table-led club & RSVP lists
  • · Underline tabs for sub-sections
  • · Mobile = collapsed sidebar (existing pattern)
V2 Mobile-first · most accessible

Focus Stack

No sidebar. Top bar + scrolling chip tabs. One thing at a time, big tap targets, sticky save bar on mobile. Reads like the public web app.

  • · Single column, max-w-3xl on desktop
  • · Chip-tab navigation (horizontal scroll)
  • · Card-based RSVPs with inline approve/reject
  • · Sticky bottom CTA bar on mobile
V3 Most visual · dashboard-first

Bento Studio

Modular tile grid. Each function is its own card. Fraunces italic for editorial flourish on identity tile. Scannable at a glance.

  • · 12-col bento grid (stacks on mobile)
  • · Dark hero tile with editorial typography
  • · Pending RSVPs surfaced inline on dashboard
  • · Analytics placeholder feels intentional
V4 Densest · power-user

Command Console

Three-pane Linear-style layout. Keyboard hints, monospaced metadata, dense rows. Built for the admin who lives here daily.

  • · Nav + list + detail tri-pane
  • · ⌘K search + J/K keyboard shortcuts
  • · Inline edit, no save/cancel modals
  • · Mobile collapses to single column
