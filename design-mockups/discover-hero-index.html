Brand fidelity

All four use the actual v1c-club-card, v1c-chip, v1c-btn-primary patterns lifted verbatim from src/app/globals.css. Same Fraunces italic display, same Oswald uppercase labels and tags, same Poppins body, same jet / bone / red / cream palette, same 0-radius sharp edges. discover-hero-system.css in this folder is the shared partial.

Recommendation

Ship V3's segmented bar for the explicit-filter mode (highest familiarity) and V2 "Ask Kip" as an entry point above it — together they cover both filter and natural-language interactions, and V2 is the brand-defensible one because it leverages /discover/smart which you've already built. V1 is the strongest "headline-first" hero if you want one unified surface. V4 works better as a future /explore page than the homepage hero — the city bento eats a lot of above-the-fold real estate, but it's the right surface once people are inside the directory.
